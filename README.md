# StarForge 🚀

**AI-powered GitHub repo booster & developer toolkit**

Generate viral READMEs, analyze star growth, create social posts, and more — all locally with Ollama or any LLM.

## Features
- ✨ One-command README generator from your code
- 📈 Star history analyzer
- 🐦 Auto-generate X/Reddit/HN posts
- 🔍 Repo health checker

## Quick Start
```bash
pip install -e .
starforge generate-readme .
```

## Why it will get stars
- Solves real pain for open-source devs
- Local-first (privacy)
- Easy to extend with agents
